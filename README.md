# 🚨 RoadSoS

> AI-Powered Smart Emergency Response System for Vehicles

## Overview

RoadSoS is an intelligent vehicle safety platform that combines AI-powered crash detection, automated SOS alerts, offline emergency routing, and real-time driver assistance to improve highway safety and emergency response.

## Features

- 🚑 AI Crash Detection
- 📡 Automatic SOS Alerts
- 📍 Offline Emergency Routing
- 🤖 Emergency Assistance Chatbot
- 📊 Vehicle Telemetry Monitoring
- 👤 Driver Profile Management
- 🎛️ Smart Vehicle Dashboard

## Tech Stack

- Python
- Flask
- Tkinter
- OpenStreetMap API
- JSON
- CSV
- Requests
- REST APIs
